实战项目名称：仿知识星球App
技术栈：前端 => uni-app （ 后端：Node.js + Mysql + Apollo + Graphql ）
已实现功能：微信登录，创建星球，内容管理，星球管理

项目实战教程地址：https://blog.csdn.net/weixin_43523043/category_11974884.html
